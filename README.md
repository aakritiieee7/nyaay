# न्याय AI (NYAAY-AI) 🏛️⚖️

## A Bias-Aware AI Legal Assistant for India

**Empowering marginalized communities with accessible legal guidance in their own language**

---

## 🌟 **The Problem We Solve**

Millions of Indians cannot access legal help due to:
- **Language Barriers** - Legal documents only in English/formal Hindi
- **Low Literacy** - Complex legal jargon is incomprehensible
- **Systemic Bias** - Discrimination based on caste, gender, economic status
- **Geographic Isolation** - No lawyers in rural areas
- **Cost Barriers** - Legal consultation is expensive

**Existing AI tools fail with:**
- Non-English or emotional speech
- Indian dialects and regional languages
- Cultural context and bias detection

---

## 🚀 **Our Solution**

### **1. Multilingual Input Processing**
- **Hindi, Hinglish, Regional Dialects** - Understands emotional speech patterns
- **Voice & Text Input** - Works for low-literacy users
- **Cultural Context** - Trained on Indian social dynamics

### **2. Intelligent Legal Mapping**
- Converts unstructured queries → structured legal categories
- Maps to relevant **Indian laws and sections**
- Provides **case precedents** and **success rates**

### **3. Built-in Bias Detection**
- Identifies **caste, gender, economic discrimination**
- Flags potential bias in legal proceedings
- Suggests **constitutional remedies**

### **4. Document Generation**
- **FIR (First Information Report)** - Police complaints
- **RTI Applications** - Right to Information requests
- **Legal Notices** - Formal legal notifications
- **Consumer Complaints** - Consumer court filings

### **5. Professional Network**
- **Verified Lawyers** - Background-checked legal professionals
- **NGOs & Legal Aid** - Free legal assistance organizations
- **Real-time Availability** - Instant connection to help

## 🚀 **Quick Start**

### **Prerequisites**
- Node.js 18+ and npm
- Python 3.8+ (for AI backend)
- Git

### **Installation**

```bash
# Clone the repository
git clone https://github.com/code-divas/nyaayai.git
cd nyaayai

# Install frontend dependencies
npm install

# Install backend dependencies (when available)
pip install -r requirements.txt

# Start development server
npm run dev

# Start backend (when available)
python app.py
```

### **Environment Variables**
```env
VITE_API_URL=http://localhost:5000
OPENAI_API_KEY=your_openai_key
FIREBASE_CONFIG=your_firebase_config
```
